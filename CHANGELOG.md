# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-09

### Added
- Initial release of Sungrow Inverter integration
- Support for SG10RT and other Sungrow models
- Modbus TCP connection with automatic discovery
- 15+ sensor entities for monitoring:
  - Solar production (daily, total)
  - Active power generation
  - Inverter temperature
  - Grid frequency
  - MPPT voltages and currents
  - Power factor and reactive power
- Number entity for export limit control
- Config flow for easy setup via Home Assistant UI
- HACS support
- Comprehensive documentation and troubleshooting guides

### Features
- Real-time solar data updates (configurable interval)
- Support for multiple inverter models
- Device grouping in Home Assistant
- Energy Dashboard integration ready
- Automatic entity discovery

### Documentation
- Prerequisite validation checklist
- Step-by-step installation guides
- Configuration examples
- Troubleshooting guide
- Diagnostic tools
